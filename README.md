# 🎲 人生模拟器 · 命运之书

> **“人生如戏，全靠选择。”**

这是一个基于 Web 的文字冒险游戏，模拟了人从呱呱坠地到生老病死的完整一生。在这里，你可以选择成为商界巨鳄、艺术大师、体育明星，或是回归田园的隐士。每一次选择，都会改变你的人生轨迹。

[**在线游玩 (点击这里)**](https://rgbrgb111.github.io/BITLIFE/)
---

## 🌟 项目特色

- **多维度养成**：智力、魅力、财富、健康、幸福、道德，六维属性自由发展。
- **天赋系统**：开局随机抽取独特天赋（如“天生聪慧”、“幸运之星”），决定人生起点。
- **海量事件**：涵盖童年、青年、中年、老年四个阶段，包含职业、情感、健康等数十个随机事件。
- **丰富结局**：根据你的选择，导向数十种不同的人生结局（如“亿万富翁”、“流浪汉”、“道德楷模”）。
- **纯前端实现**：仅使用 HTML, CSS, JavaScript，无需后端服务器，开箱即用。

## 🚀 如何游玩

1. 点击上方的在线游玩链接，或下载代码在本地运行。
2. 选择你的天赋。
3. 根据剧情提示，做出你的选择。
4. 观察属性变化，走向你独一无二的命运。

## 💾 本地运行 (开发模式)

如果你想要修改代码或在本地调试，请使用以下方式启动，**不要直接双击 HTML 文件**（会因浏览器安全策略导致报错）：

```bash
# 1. 克隆项目
git clone https://github.com/rgbrgb111/BITLIFE.git
cd BITLIFE

# 2. 使用 Python 快速启动 (需安装 Python)
python -m http.server 8000

# 或使用 Node.js (需安装 nodejs)
npx http-server
